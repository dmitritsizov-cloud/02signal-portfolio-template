# 02Signal portfolio — Dmitri Tsizov

> **02Signal — Äri automatiseerimine tehisaru (AI) abil.** Ettevõtluskeskus OÜ, 2025–2026.

Selle portfoolio eesmärk on näidata, **mida ma õppisin nädala kaupa** ja **mida ma tegelikult ehitasin**, mitte ainult mida kuulasin. Iga nädal lõpeb ühe artefaktiga, mida saab demonstreerida ja mõõta.

---

## Mina kui ehitaja

Olen ettevõtja vaatega õppija. Hindan **kiiret minimaalset võitu** rohkem kui täiuslikku plaani. Igal nädalal küsin endalt:

1. Mis on **otsuse küsimus**, millele ma vastan?
2. Mis on **andmeallikas**, mida ma kasutan?
3. Mida **inimene peab kontrollima**, enne kui AI väljundit usaldatakse?
